# 飞书交互规范

## 1. 用户主动分析卡片

### D/C 级

按钮：

```text
[加入长期产品池] [忽略]
```

展示内容：

- 一句话结论
- 等级和置信度
- 最多 3-5 条无机会原因
- 下次触发条件
- 关键边界

禁止：

- 展示“加入机会监控”
- 展示“加入模拟盘”
- 自动加入观察列表

### B 级

按钮：

```text
[加入机会监控] [加入长期产品池] [忽略]
```

### A/S 级且 trade_plan 完整且风控通过

按钮：

```text
[加入模拟盘] [加入机会监控] [加入长期产品池] [忽略]
```

### A/S 级但 trade_plan 不完整

按钮：

```text
[加入机会监控] [加入长期产品池] [忽略]
```

## 2. 实时预警

只在以下事件推送：

- 模拟盘开仓
- 模拟盘平仓
- 止损
- 止盈
- 调整止损
- 强风控警报
- 账户回撤警报
- 机会监控触发并形成可执行计划

普通 scheduled analysis 不实时刷屏。

## 3. 每小时播报

必须是摘要，不是逐币完整分析。

必须包含：

1. 系统状态
2. 模拟盘摘要
3. S/A/B 高等级机会
4. 当前机会监控
5. C/D 无优势品种汇总
6. 风险事件
7. 存储组件状态：Redis / SQLite / Parquet / DuckDB

## 4. 详细分析

只有用户发送：

```text
详细分析 ETHUSDT
```

才展示模块明细。

## 5. 研判来源

禁止：

```text
研判来源：LLM/GA Agent
```

必须：

```text
研判来源：动态 Skill 工具计算 + GA Master Controller 综合
引用：price_action_log_id、momentum_log_id、trend_stage_log_id、smc_log_id、chanlun_log_id、ga_decision_id
```
