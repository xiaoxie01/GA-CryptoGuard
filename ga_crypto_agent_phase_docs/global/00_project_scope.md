# 00 项目范围

## 系统目标

构建一个可长期运行的 GA CryptoGuard：

- 默认分析主流币安 USDT-M 合约品种。
- 用户可通过飞书自然语言添加、暂停、移除监控品种。
- 用户可临时分析指定品种。
- 分析后可选择加入模拟盘、加入机会监控、加入长期产品池或忽略。
- 定时按照 UTC-0 获取并总结 K 线数据。
- 15m 定时生成市场状态。
- 3m/5m 更新模拟盘收益。
- 平仓后自动复盘。
- 根据复盘生成候选策略补丁。
- 候选策略必须通过影子测试后才能升级。

## 非目标

- 不接实盘。
- 不做真实下单。
- 不保存交易权限 API Key。
- 不承诺收益。
- 不让 LLM 直接用原始 tick 做裸判断。
- 不让定时任务占用用户对话 Loop。
