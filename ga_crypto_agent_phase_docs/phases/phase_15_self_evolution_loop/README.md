# Phase 15：自进化闭环

## 阶段目标

自进化闭环 是 GA CryptoGuard 在 MVP 后推进路线中的第 15 阶段。

本阶段只实现当前范围内的功能，不提前实现后续阶段高级能力。

## 前置条件

- 已阅读 `global/` 全局设计。
- 已执行 `global/schema.sql` 基础建表或对应迁移。
- 前置阶段已通过验收。
- 系统仍保持“不接实盘”的强制边界。

## 本阶段主要任务

- 复盘归因聚合
- 生成策略补丁
- 影子测试验证
- 升级 active
- 沉淀 strategy_memory

## 本阶段完成后应达到

- 自进化全链路可审计
- 策略升级基于样本和指标
- 不能过拟合单一品种
- 系统可解释为何升级或拒绝
