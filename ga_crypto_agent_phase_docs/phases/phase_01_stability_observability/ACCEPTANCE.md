# Phase 01 验收标准

本阶段必须满足以下验收项：

- [ ] 连续运行 72 小时无致命错误
- [ ] 后台 15m 任务运行时用户消息仍能优先响应
- [ ] 所有定时任务都有 scheduler_runs 记录
- [ ] 重复飞书事件不会重复处理

## 通用验收

- [ ] 所有新增测试通过。
- [ ] 数据库迁移可重复执行。
- [ ] 日志中无未处理异常。
- [ ] 飞书用户消息不被后台任务阻塞。
- [ ] 后台任务不会写入用户短期会话上下文。
- [ ] 所有时间均为 UTC。
- [ ] 不存在真实下单逻辑。
- [ ] 所有新增功能有失败分支处理。
- [ ] 所有关键数据可追溯。
