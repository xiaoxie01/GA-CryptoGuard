# Phase 05：模拟盘执行质量

## 阶段目标

模拟盘执行质量 是 GA CryptoGuard 在 MVP 后推进路线中的第 5 阶段。

本阶段只实现当前范围内的功能，不提前实现后续阶段高级能力。

## 前置条件

- 已阅读 `global/` 全局设计。
- 已执行 `global/schema.sql` 基础建表或对应迁移。
- 前置阶段已通过验收。
- 系统仍保持“不接实盘”的强制边界。

## 本阶段主要任务

- 计算 MFE/MAE
- 计算入场效率/出场效率
- 计算信号衰减
- 记录止损止盈路径
- 模拟盘权益曲线快照

## 本阶段完成后应达到

- 每笔平仓 trade 都有 MFE/MAE
- 止盈止损触发准确
- 收益曲线每 3/5 分钟更新
- 大回撤可触发提醒
