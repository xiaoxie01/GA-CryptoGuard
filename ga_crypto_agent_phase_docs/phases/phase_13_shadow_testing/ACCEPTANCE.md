# Phase 13 验收标准

本阶段必须满足以下验收项：

- [ ] candidate 不影响用户推送
- [ ] 样本不足不能升级
- [ ] 对比指标包含 avg_r/win_rate/drawdown
- [ ] 升级必须人工确认或配置允许

## 通用验收

- [ ] 所有新增测试通过。
- [ ] 数据库迁移可重复执行。
- [ ] 日志中无未处理异常。
- [ ] 飞书用户消息不被后台任务阻塞。
- [ ] 后台任务不会写入用户短期会话上下文。
- [ ] 所有时间均为 UTC。
- [ ] 不存在真实下单逻辑。
- [ ] 所有新增功能有失败分支处理。
- [ ] 所有关键数据可追溯。
