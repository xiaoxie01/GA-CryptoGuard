# Phase 08：趋势阶段模块

## 阶段目标

趋势阶段模块 是 GA CryptoGuard 在 MVP 后推进路线中的第 8 阶段。

本阶段只实现当前范围内的功能，不提前实现后续阶段高级能力。

## 前置条件

- 已阅读 `global/` 全局设计。
- 已执行 `global/schema.sql` 基础建表或对应迁移。
- 前置阶段已通过验收。
- 系统仍保持“不接实盘”的强制边界。

## 本阶段主要任务

- 实现 early/middle/late/range/transition
- 融合 1D/4H/1H/15m
- 趋势阶段影响策略选择与信号降级
- 记录阶段变化事件

## 本阶段完成后应达到

- 每次分析输出 trend_stage
- late 阶段追单被降级
- range 阶段趋势策略被过滤
- 阶段变化可推送飞书
