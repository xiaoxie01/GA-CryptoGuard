# Phase 02 验收标准

本阶段必须满足以下验收项：

- [ ] 任意 signal 可追溯到 snapshot
- [ ] 任意 paper_trade 可追溯到 signal 和 snapshot
- [ ] 分析只使用已收盘 K 线
- [ ] 缺失数据会标记 data_quality 而不是静默通过

## 通用验收

- [ ] 所有新增测试通过。
- [ ] 数据库迁移可重复执行。
- [ ] 日志中无未处理异常。
- [ ] 飞书用户消息不被后台任务阻塞。
- [ ] 后台任务不会写入用户短期会话上下文。
- [ ] 所有时间均为 UTC。
- [ ] 不存在真实下单逻辑。
- [ ] 所有新增功能有失败分支处理。
- [ ] 所有关键数据可追溯。
